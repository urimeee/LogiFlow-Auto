# ✏️ 수동 주문 입력 기능 가이드

## 📌 개요
**버전**: v3.19  
**출시일**: 2026-03-22  
**기능**: 시딩/CS 주문을 직접 입력하여 3PL 형식으로 자동 변환

## 🎯 주요 기능

### 1. 카테고리 선택
- **시딩**: 샘플 발송, 홍보 목적 주문
- **CS**: 고객 서비스 관련 주문 (교환, 반품 재발송 등)

### 2. 주문번호 자동 생성
#### 시딩 주문번호 규칙:
```
{오늘날짜}_Seed_{고유번호}

예시:
- 20260322_Seed_a (첫 번째 시딩 주문)
- 20260322_Seed_b (두 번째 시딩 주문)
- 20260322_Seed_c (세 번째 시딩 주문)
```

#### CS 주문번호 규칙:
```
{오늘날짜}_CS_{고유번호}

예시:
- 20260322_CS_a (첫 번째 CS 주문)
- 20260322_CS_b (두 번째 CS 주문)
- 20260322_CS_c (세 번째 CS 주문)
```

### 3. 입력 필드
#### 필수 정보:
- ✅ **주문자명**: 받는 사람 이름
- ✅ **전화번호**: 연락처 (예: 010-1234-5678)
- ✅ **주소**: 배송 주소
- ✅ **상품 코드**: 마스터 코드에서 자동으로 불러온 목록에서 선택

#### 선택 정보:
- **수량**: 기본값 1개
- **배송 메세지**: 배송 요청사항 (선택)

## 🚀 사용 방법

### Step 1: 마스터 코드 업로드
1. 사이드바에서 **물류_코드명.xlsx** 파일 업로드
2. "✅ 마스터 코드 데이터 로드 완료" 메시지 확인

### Step 2: 수동 입력 탭 선택
1. 상단 탭에서 **"✏️ 수동 입력"** 클릭
2. 마스터 코드가 로드되어 있는지 확인

### Step 3: 주문 정보 입력
1. **카테고리 선택**: 시딩 또는 CS
2. **주문자명 입력**: 받는 사람 이름
3. **전화번호 입력**: 010-1234-5678 형식
4. **주소 입력**: 전체 배송 주소
5. **상품 코드 선택**: 드롭다운에서 선택
6. **수량 입력**: 필요한 수량 (기본 1개)
7. **배송 메세지 입력** (선택사항)

### Step 4: 주문 추가
1. **"➕ 주문 추가"** 버튼 클릭
2. 주문번호 자동 생성 확인
3. 추가된 주문 목록에서 확인

### Step 5: 통합 처리
1. 파일 업로드 주문과 함께 자동 통합
2. "📥 통합 파일 다운로드" 버튼으로 엑셀 파일 다운로드
3. 수동 입력 주문도 32개 표준 3PL 컬럼으로 변환됨

## 💡 주요 특징

### ✅ 자동화 기능
- **주문번호 자동 생성**: 날짜 + 카테고리 + 고유번호
- **상품 정보 자동 매칭**: 마스터 코드에서 상품명, 옵션명 자동 입력
- **3PL 형식 자동 변환**: 32개 표준 컬럼으로 자동 변환

### ✅ 편의 기능
- **실시간 목록 확인**: 추가한 주문을 즉시 확인
- **일괄 삭제**: "🗑️ 모든 수동 주문 삭제" 버튼
- **통합 다운로드**: 파일 업로드 주문과 함께 통합

### ✅ 검증 기능
- **필수 항목 체크**: 주문자명, 전화번호, 주소, 상품 코드 누락 시 경고
- **상품 코드 검증**: 마스터 코드에 없는 상품 코드 입력 시 오류 메시지

## 📊 출력 형식

### 3PL 32개 표준 컬럼
수동 입력 주문도 다음 형식으로 자동 변환됩니다:

| 컬럼 | 값 |
|------|-----|
| 쇼핑몰 코드 | ONLINE1 |
| 쇼핑몰 이름 | 시딩 또는 CS |
| 쇼핑몰 묶음 배송 번호 | 주문번호와 동일 |
| 쇼핑몰 주문번호 | 예: 20260322_Seed_a |
| 수취인 명 | 입력한 주문자명 |
| 수취인 전화번호 | 입력한 전화번호 |
| 수취인 기본 주소 | 입력한 주소 |
| 쇼핑몰 상품 코드 | 선택한 상품 코드 |
| 쇼핑몰 상품 이름 | 마스터 코드에서 자동 매칭 |
| 쇼핑몰 옵션 이름 | 마스터 코드에서 자동 매칭 |
| 주문 수량 | 입력한 수량 |
| 배송 메세지 | 입력한 배송 메세지 |
| 매칭 방법 | 수동 입력 |
| 확인 필요 | False (수동 입력은 확인 불필요) |

## ⚠️ 주의사항

### 1. 마스터 코드 필수
- 수동 입력 기능을 사용하기 전에 반드시 마스터 코드 파일을 업로드해야 합니다
- 마스터 코드가 없으면 상품 정보 매칭이 불가능합니다

### 2. 상품 코드 정확성
- 드롭다운에서 올바른 상품 코드를 선택하세요
- 마스터 코드에 없는 상품은 선택할 수 없습니다

### 3. 주문번호 중복 방지
- 같은 날짜에 여러 개의 시딩/CS 주문을 입력하면 자동으로 a, b, c... 순서로 고유번호가 부여됩니다
- 수동으로 주문번호를 변경할 수 없습니다

### 4. 세션 상태 관리
- 추가한 주문은 브라우저 세션에 저장됩니다
- 페이지를 새로고침하거나 닫으면 저장되지 않은 주문은 사라집니다
- 반드시 "📥 통합 파일 다운로드"로 저장하세요

## 🎓 사용 예시

### 예시 1: 인플루언서 시딩 발송
```
카테고리: 시딩
주문자명: 홍길동
전화번호: 010-1234-5678
주소: 서울시 강남구 테헤란로 123
상품 코드: DA-DI-PW (딥 페리윙클)
수량: 1
배송 메세지: 문 앞에 놓아주세요

→ 주문번호: 20260322_Seed_a
```

### 예시 2: 고객 CS 재발송
```
카테고리: CS
주문자명: 김철수
전화번호: 010-9876-5432
주소: 부산시 해운대구 센텀중앙로 456
상품 코드: C-AR-GRK-03 (극락 3개)
수량: 1
배송 메세지: 불량품 교환 - 빠른 배송 부탁드립니다

→ 주문번호: 20260322_CS_a
```
